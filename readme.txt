The program is called "Otzovik". 
The users can see the reviews of other users, they can leave his own review, see only positive / negative reviews.

Classes:

Main - standard "Main", where the program starts from.

Application - from there, we request the actions that the program will perform.

Review - creating and displaying reviews

DB, IDB - database connection

Review_Rep, IReview_Rep - work with the database: display all reviews, create a review, show positive / negative reviews
